# HR Analytics Dashboard

## Project Overview
This project analyzes employee compensation and workforce distribution using HR data.

The objective was to transform raw HR data into actionable business insights that support data-driven decision-making.

## Workflow
- Data Cleaning: Excel
- Exploratory Data Analysis (EDA): Python (Pandas, Matplotlib, Seaborn)
- Dashboard Development: Power BI

## Key Analysis
- Total compensation by department and job role
- Workforce distribution across departments
- Compensation split by pay type
- Average hourly rate by job role
- HR KPIs: total employees, average salary, average hourly rate

## Key Insights
- Compensation is primarily salary-driven.
- Certain departments contribute significantly to total organizational cost.
- Workforce concentration varies across departments.
- Higher hourly rates strongly influence overall compensation.

## Tools Used
Power BI | Python | Pandas | Matplotlib | Seaborn | Excel
